# CP Password Flipper

Password eye flipper to view passwords in EE4 and 5

## Installation

1. Copy the `cp_password_flipper` directory to your `system/user/addons` folder.
2. Copy the `themes/cp_password_flipper` directory to your `themes/user` folder.
